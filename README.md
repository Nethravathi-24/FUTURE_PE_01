# AI Website Copy Generator for Local Businesses

## Prompt Engineering Internship Task 1

This project demonstrates how structured prompt engineering can be used to generate high-quality website copy for local businesses using AI tools.

The system focuses on designing and optimizing prompts that produce clear, persuasive, and structured website content suitable for real-world business websites.

---

## Project Objective

Many small businesses struggle to create professional website content.  
This project explores how AI prompt engineering can help generate structured and conversion-focused website copy.

The example business used in this project is:

SmileCraft – Multi-Specialty Dental Clinic located in Bangalore.

---

## Key Features

- Structured prompt engineering workflow
- Iterative prompt optimization (V1 → V2 → V3)
- AI-generated website homepage copy
- Business-focused content structure
- Simple prompt generator demo interface

---

## Prompt Engineering Approach

The prompt was improved through multiple iterations:

### Version 1
Basic prompt with minimal instructions.

### Version 2
Added business context including:
- Target audience
- Services
- Tone
- Structured output format

### Version 3
Final optimized prompt with:
- Trust signals
- Local authority
- Conversion-focused messaging
- Credibility indicators

This iterative process improved the quality and realism of the generated website copy.
