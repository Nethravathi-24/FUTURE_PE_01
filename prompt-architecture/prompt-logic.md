# Prompt Engineering Logic

This project demonstrates how structured prompt engineering improves AI-generated content quality.

## 1. Role Assignment
The AI is instructed to act as a professional website copywriter.  
Assigning a role helps guide the tone, expertise level, and writing style of the output.

## 2. Context Enrichment
Providing information such as business name, location, target audience, and services helps the AI generate more relevant and accurate content.

## 3. Structured Output
The prompt clearly specifies sections such as Hero, About, Services, and CTA.  
This ensures that the generated output follows a consistent website structure.

## 4. Iterative Optimization
The prompt was improved through three iterations:

V1 → Basic prompt with minimal instructions  
V2 → Added business details and structure  
V3 → Added trust signals and conversion-focused elements

Each iteration improved the quality and usefulness of the generated output.

## 5. Conversion Focus
The final prompt emphasizes persuasive messaging and trust-building elements, which are essential for business websites.
